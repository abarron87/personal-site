--- layout: work title: copy tool description: A tool to allow the previewing of copy text on banners of various dimensions in real-time. type: work repo-name: copy-tool ---

Details

  • It leverages the awesomeness of the two-way-data-binding and filtering (among other features) of Angular to update all visible banners with text changes on the fly.
  • Both plain text and HTML can be entered, and the container <h3> element can be overridden from a simple drop down.
  • There are 2 input options:
    • One for long copy which will go on the larger banners.
    • One for short copy.
  • The final code can then be taken by clicking "Grab" by individual banners or the "Grab all" button at the top.

Demo

A demo can be viewed at https://copy-tool.herokuapp.com/.
